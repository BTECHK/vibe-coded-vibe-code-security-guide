## Methodology

This guide was built through:

1. **Research** — Gathered sources from security experts, platform documentation, and community resources
2. **Synthesis** — Combined insights from multiple sources into a unified checklist
3. **LLM Iteration** — Pressure-tested coverage by iterating with Claude, Gemini, and ChatGPT (asking each to review as a "principal security engineer")
4. **Consolidation** — Removed redundancy, kept the most actionable and relevant items
